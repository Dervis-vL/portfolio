3D Bridge Builder Web App

Back

I conceptualized a plan to develop a web app aimed at facilitating input data for the Traffic Bridge Tool.

This web app provides a user interface displaying real-time input parameters within a 3D bridge visualization using Unity.

The backend operates on a microservices architecture. One service integrates the Traffic Bridge Tool, while another estimates construction costs based on input data, and yet another calculates the Environmental Cost Indicator (MKI) score.

The frontend dynamically presents construction costs and MKI scores. The app communicates results with ANT CDE.

The project aims to bridge the gap between clients and contractors by simplifying bridge design.

It empowers government agencies or architects to provide draft designs early on, expediting permit applications.

Within this endeavor, I conceptualized the microservices architecture, authored the project plan, and serve as the product owner during development.

The web app is currently in the development phase.

Technologies: PYTHON, JIRA, SCRUM, GITLAB, POETRY, VSCODE, GIT
